### Prerequisites

You should have `nodejs` installed. If you don't install it from [here](https://nodejs.org/en/download/).

### Installing

1. Clone the project

2. Run this command to download npm packages
   ```sh
   npm install
   ```
3. Run this command to download npm packages

   ```sh
   npm start
   ```
